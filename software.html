What is it?

Software development refers to using computer code to create, design, deploy and support computer software. Computer software applications range from productivity software to create documents, gaming software to play video games, and video conferencing software to stay connected at school. Computer systems refer to the hardware, software, and networking systems that an organization uses to generate information and be productive.
